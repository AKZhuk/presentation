What is HTML Canvas?
The HTML canvas element is used to draw graphics, on the fly, via JavaScript methods.
The canvas element is only a container for graphics. You must use JavaScript to actually draw something.
Canvas has several methods for drawing paths, boxes, circles, text, and adding images.
History of HTML canvas
Canvas was initially introduced by Apple for use in their own Mac OS X WebKit component in 2004.
Later, in 2005 it was adopted in version 1.8 of Gecko browsers, and Opera in 2006, and standardized by the Web Hypertext Application Technology Working Group (WHATWG) on new proposed specifications for next generation web technologies.

HTML Canvas Drawing

Step 1: Find the Canvas Element

First of all, you must find the canvas element.
This is done by using the HTML DOM method getElementById(): 

							var canvas = document.getElementById("rectangle");
							var ctx = canvas.getContext("2d");
							ctx.fillStyle = "#FF0000";
							ctx.fillRect(50,10,100,80);
							ctx.clearRect(70,20,60,60);
							ctx.strokeStyle = "#FF0000";
							ctx.strokeRect(75,25,50,50);

Step 2: Create a Drawing Object

Secondly, you need a drawing object for the canvas.
The getContext() is a built-in HTML object, with properties and methods for drawing: 

							var canvas = document.getElementById("rectangle");
							var ctx = canvas.getContext("2d");
							ctx.fillStyle = "#FF0000";
							ctx.fillRect(50,10,100,80);
							ctx.clearRect(70,20,60,60);
							ctx.strokeStyle = "#FF0000";
							ctx.strokeRect(75,25,50,50);

Step 3: Draw on the Canvas

    Thirdly, you can draw on the canvas:
  • Set the fill style of the drawing object to the color red. The fillStyle property can be a CSS color, a gradient, or a pattern. The default fillStyle is black.
  • The fillRect(x,y,width,height) method draws a rectangle, filled with the fill style, on the canvas: 

							var canvas = document.getElementById("rectangle");
							var ctx = canvas.getContext("2d");
							ctx.fillStyle = "#FF0000";
							ctx.fillRect(50,10,100,80);
							ctx.clearRect(70,20,60,60);
							ctx.strokeStyle = "#FF0000";
							ctx.strokeRect(75,25,50,50);

Step 4: Clear the Canvas

    You can clear rectangle area on the canvas:
  • The clearRect(x,y,width,height) method clear a rectangle area on the canvas: 

							var canvas = document.getElementById("rectangle");
							var ctx = canvas.getContext("2d");
							ctx.fillStyle = "#FF0000";
							ctx.fillRect(50,10,100,80);
							ctx.clearRect(70,20,60,60);
							ctx.strokeStyle = "#FF0000";
							ctx.strokeRect(75,25,50,50);

Step 5: Draw on the Canvas

    Finally, you can draw stroke rectangle on the canvas:
  • Set the stroke style of the drawing object to the color red. The default strokeStyle is black.
  • The strokeRect(x,y,width,height) method draws a rectangle, with border colored the stroke style, on the canvas: 

							var canvas = document.getElementById("rectangle");
							var ctx = canvas.getContext("2d");
							ctx.fillStyle = "#FF0000";
							ctx.fillRect(50,10,100,80);
							ctx.clearRect(70,20,60,60);
							ctx.strokeStyle = "#FF0000";
							ctx.strokeRect(75,25,50,50);

HTML Canvas Coordinates
  • The HTML canvas is a two-dimensional grid.
  • The upper-left corner of the canvas has the coordinates (0,0)
  • In the previous chapter we used method fillRect(50,10,100,80).This means: Start at the point (50,10) and draw a rectangle with width 100 and height 80 pixels.

Draw a line

To draw a straight line on a canvas, use the following methods:

  • moveTo(x,y) - defines the starting point of the line
  • lineTo(x,y) - defines the ending point of the line
  • use one of the "ink" methods, like stroke() - which strokes (outlines) the current or given path with the current stroke style
As example, we define a starting point in position (0,0), and an ending point in position (200,100). Then use the stroke() method to actually draw the line: 

							var canvas = document.getElementById("myCanvas");
							var ctx = canvas.getContext("2d");
							ctx.moveTo(0, 0);
							ctx.lineTo(200, 100);
							ctx.stroke();
As a result we get line from point (x: 0, y: 0) to (x: 200, y: 100):

Draw a circle

To draw a circle on a canvas, use the following methods:

  • beginPath() - begins a path
  • arc(x,y,r,startangle,endangle) - creates an arc/curve. To create a circle: set startangle to 0, endangle to 2*Math.PI, x and y define the center of the circle, r parameter defines the radius of the circle.
  • use stroke() - which strokes (outlines) the current or given path with the current stroke style
As example, we define a circle with the arc() method. Then use the stroke() method to actually draw the circle: 

							var canvas = document.getElementById("myCanvas");
							var ctx = canvas.getContext("2d");
							ctx.beginPath();
							ctx.arc(95, 50, 40, 0, 2 * Math.PI);
							ctx.stroke();
As a result we get circle with center at (x: 95, y: 40) with 40 pixels radius:

Canvas Gradients

Gradients can be used to fill rectangles, circles, lines, text, etc. Shapes on the canvas are not limited to solid colors.

    There are two different methods to define gradients:

  • createLinearGradient(x,y,x1,y1) - creates a linear gradient.
  • createRadialGradient(x,y,r,x1,y1,r1) - creates a radial/circular gradient.
Once we have a gradient object, we must add two or more color stops. The addColorStop() method specifies the color stops, and its position along the gradient. Gradient positions can be anywhere between 0 to 1.
Examle: createLinearGradient() 

							var c = document.getElementById("myCanvas");
							var ctx = c.getContext("2d");

							// Create gradient
							var grd = ctx.createLinearGradient(0,0,200,0);
							grd.addColorStop(0.1,"white");
							grd.addColorStop(0.4,"red");
							grd.addColorStop(0.7,"white");


							// Fill with gradient
							ctx.fillStyle = grd;
							ctx.fillRect(10,10,150,80);
As a result we get rectangle with next gradient:
Examle: createRadialGradient() 

							var c = document.getElementById("myCanvas");
							var ctx = c.getContext("2d");

							// Create gradient
							var grd = ctx.createRadialGradient(75,50,5,90,60,100);
							grd.addColorStop(0.1,"red");
							grd.addColorStop(0.4,"yellow");
							grd.addColorStop(0.7,"blue");

							// Fill with gradient
							ctx.fillStyle = grd;
							ctx.fillRect(10,10,150,80);
As a result we get rectangle with next gradient:

Canvas Text

    To draw text on a canvas we can use next properties and methods:

  • font - defines the font properties for the text.
  • fillText(text,x,y) - draws "filled" text on the canvas.
  • strokeText(text,x,y) - draws text on the canvas (no fill).
Example: draw text using 'stroke' and 'fill' text methods 

							var canvas = document.getElementById("myCanvas");
							var ctx = canvas.getContext("2d");
							ctx.font = "30px Arial";
							ctx.strokeText("Hello World",10,50);
							ctx.fillText("Hello World",10,80);
As a result we get 'stroked' and 'filled' text:
Example: change color, font and text align 

							var canvas = document.getElementById("myCanvas");
							var ctx = canvas.getContext("2d");
							ctx.font="30px Comic Sans MS";
							ctx.fillStyle = "red";
							ctx.textAlign = "center";
							ctx.fillText("Hello World", canvas.width/2, canvas.height/2);
We use new method textAlign(), which sets or returns the current alignment for text content.
As a result we get red text, with 30px Comic Sans MS font with center align:

Canvas Images

    To draw an image on a canvas we can use method:

    
							drawImage(image, sx, sy, sWidth, sHeight, dx, dy, dWidth, dHeight);
  1. image - an element to draw into the context. The specification permits any canvas image source.
  2. sx, sy - the x and y -axis coordinate of the top left corner of the sub-rectangle of the source image.
  1. sWidth, sHeight -the width and height of the sub-rectangle of the source image.
  2. dx, dy - the x and y -axis coordinate in the destination canvas at which to place the top-left corner of the source image.
  3. dWidth, dHeight - the width and height to draw the image in the destination canvas. This allows scaling of the drawn image. If not specified, the image is not scaled in width when drawn.
